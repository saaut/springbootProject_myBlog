# 📝 My Blog



이 프로젝트는 기본적인 게시판 프로젝트입니다. Spring Boot를 학습하고 전체적인 백엔드 흐름을 이해하기 위해 만들어졌습니다.

---

## 🚀 주요 기능
- CRUD 기능을 갖춘 게시판
- OAuth2 Google 로그인

---

## 🛠️ 사용된 기술
### 주요 프레임워크/라이브러리
#### 백엔드
- Java17
- SpringBoot 3
- JPA
- OAuth 2.0

#### 빌드 도구
- Gradle

#### 데이터베이스
- ORM

### 프론트엔드
- HTML
- JavaScript
- Thymeleaf

### 배포
- AWS Elastic Beanstalk

### CI/CD
- GitHub Action
